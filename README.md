# Suleman Kondkari | Cinematic Portfolio 🎬

A high-performance, editorial-grade portfolio built for maximum visual impact. This is not just a project; it's a digital experience designed with a cinematic red-and-black aesthetic, smooth motion curves, and high-quality storytelling.

## ✨ Living the Experience

This portfolio is a showcase of cutting-edge web technologies integrated into a seamless, high-performance interface.

- **Cinematic Visuals**: Deep black themes with bold editorial typography.
- **GSAP & Framer Motion**: Advanced scroll-driven animations and perspective shifts.
- **High-Performance Assets**: Integrated video showreels and optimized media.
- **Mobile First**: Fully responsive across all devices without compromising the aesthetic.

## 🛠️ The Engine Room

Built with the modern stack for speed and smooth interactions:

- **Framework**: [Next.js 15](https://nextjs.org/) (App Router)
- **Design**: [Tailwind CSS](https://tailwindcss.com/) + Custom Design System
- **Motion**: [GSAP](https://gsap.com/) & [Framer Motion](https://www.framer.com/motion/)
- **Components**: [Shadcn UI](https://ui.shadcn.com/) & Magic UI
- **Workflow**: Automated asset hydration and SEO optimized

## 🚀 Getting Started

If you want to explore the code behind the curtain:

1. **Clone the Repo:**
   ```bash
   git clone https://github.com/SulemanKondkari/portfolioo.git
   ```

2. **Install Dependencies:**
   ```bash
   npm install
   ```

3. **Set Up Keys:**
   Create a `.env.local` and add your Web3Forms access key:
   ```env
   NEXT_PUBLIC_WEB3FORMS_KEY=your_access_key
   ```

4. **Launch:**
   ```bash
   npm run dev
   ```

## 📂 Project Structure

- `src/app`: Page architecture and routing.
- `src/components`: Premium UI components including Macbook scrolls and layered text.
- `public/`: High-resolution assets, videos, and project ZIPs.
